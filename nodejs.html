1. Qu'est-ce que Node.js ?

Node.js est un environnement qui permet d’exécuter du code JavaScript en dehors du navigateur.

Il utilise le moteur V8 de Chrome et permet de :

  • Lire/écrire des fichiers
  • Créer un serveur web
  • Communiquer avec une base de données
  • Exécuter des scripts automatisés
Pas de DOM, pas de window → mais plein de nouveaux outils natifs !

2. Installation & Mise à Jour sur les 3 OS

Node.js est disponible sur Windows, macOS et Linux. Voici les méthodes officielles et fiables.

Windows

  1. Aller sur nodejs.org
  2. Cliquer sur Windows Installer (.msi) → version LTS
  3. Lancer le fichier téléchargé
  4. Cocher : "Add to PATH" (important !)
  5. Suivant → Installer → Terminer
node --version
npm --version

macOS

  1. Aller sur nodejs.org
  2. Cliquer sur macOS Installer (.pkg) → version LTS
  3. Ouvrir le fichier .pkg
  4. Suivre l’assistant → Continuer → Installer
  5. Mot de passe admin si demandé
node --version
npm --version

Linux (Ubuntu/Debian)

Méthode officielle avec NodeSource (recommandée)

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

Ou via snap :

sudo snap install node --classic
node --version
npm --version
Vérification finale (tous les OS) :
Ouvre un terminal → tape :
node -v && npm -v
→ Doit afficher les versions (ex: v20.17.0 et 10.8.1)

3. Mise à Jour de Node.js

Node.js sort une nouvelle version LTS tous les 6 mois. Voici comment mettre à jour sans tout casser.

Windows & macOS

  1. Télécharge la nouvelle version LTS sur nodejs.org
  2. Lance l’installateur
  3. Il remplace automatiquement l’ancienne version
  4. Redémarre ton terminal
node -v → v22.9.0 (nouvelle version)

Linux (NodeSource)

# Mise à jour vers la dernière LTS curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get upgrade -y nodejs

Ou avec nvm (recommandé pour devs) :

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install --lts
nvm use --lts
Avec nvm (Node Version Manager) → tu peux avoir plusieurs versions en parallèle :
nvm install 18 → Node 18
nvm install 20 → Node 20
nvm use 20 → basculer

4. Premier Script Node.js

Étape 1 : Créer un dossier

mkdir mon-premier-node
cd mon-premier-node

Étape 2 : Créer un fichier

Créer hello.js :

// hello.js
console.log("Bonjour depuis Node.js !");
console.log("Nous sommes le", new Date().toLocaleDateString());

Étape 3 : Exécuter

node hello.js

Résultat :

Bonjour depuis Node.js ! Nous sommes le 16/11/2025

5. Les Modules Natifs (sans installation)

Module fs — Fichiers

Créer lire.js :

// lire.js
import fs from 'fs';
try {
    const contenu = fs.readFileSync('hello.js', 'utf8');
    console.log("Contenu de hello.js :\n");
    console.log(contenu);
} catch (err) {
    console.log("Erreur :", err.message);
}
node lire.js

Écriture de fichier

Créer ecrire.js :

// ecrire.js
import fs from 'fs';
const message = `Fichier généré le ${new Date().toLocaleString()}\n`;
fs.writeFileSync('sortie.txt', message);
console.log("Fichier 'sortie.txt' créé !");
node ecrire.js

6. Créer un Serveur Web (sans Express)

Créer server.js

// server.js
import http from 'http';
const server = http.createServer((req, res) => {
    res.writeHead(200, { 'Content-Type': 'text/html; charset=utf-8' });
    res.write('<h1>Mon Premier Serveur Node.js !</h1>');
    res.end();
});
server.listen(3000, () => {
    console.log('Serveur sur http://localhost:3000');
});
node server.js

Ouvrir localhost:3000
