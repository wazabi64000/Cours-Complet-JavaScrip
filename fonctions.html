1. Déclarations et Expressions de Fonctions

Déclaration de Fonction

Hoistée (déplacée en haut du scope). Disponible avant sa déclaration.

function saluer(nom) {
    return `Bonjour ${nom}`;
}
console.log(saluer('Marie')); // Fonctionne

Expression de Fonction

Non hoistée. Doit être définie avant utilisation.

const saluer = function(nom) {
    return `Bonjour ${nom}`;
};
console.log(saluer('Marie'));

Fonction Fléchée (Arrow Function)

Syntaxe concise. Pas de this propre, pas de arguments.

const saluer = (nom) => `Bonjour ${nom}`;
const addition = (a, b) => a + b;
const carre = x => x * x;

2. Paramètres et Valeurs par Défaut

Paramètres par Défaut

function creerUtilisateur(nom = 'Anonyme', age = 18) {
    return { nom, age };
}

Paramètre Rest

Collecte les arguments restants dans un tableau.

function somme(...nombres) {
    return nombres.reduce((total, n) => total + n, 0);
}
somme(1, 2, 3, 4); // 10

Paramètre Spread

Décompose un tableau ou objet en arguments.

const nombres = [1, 2, 3];
Math.max(...nombres); // 3

3. Portée (Scope) et Closures

Scope Global vs Local

let global = 'global';

function test() {
    let local = 'local';
    console.log(global); // accessible
    console.log(local);  // accessible
}
console.log(local); // Erreur

Block Scope (let/const)

if (true) {
    let bloc = 'dans le bloc';
}
console.log(bloc); // Erreur

Closures

Fonction qui capture son environnement lexical.

function creerCompteur() {
    let count = 0;
    return function() {
        return ++count;
    };
}
const compteur = creerCompteur();
compteur(); // 1
compteur(); // 2

4. Fonctions d'Ordre Supérieur

Fonctions qui prennent ou retournent d'autres fonctions.

Callback

function executerCallback(cb) {
    cb();
}
executerCallback(() => console.log('Callback exécuté'));

Méthodes d'Array

  • map() : Transformation
  • filter() : Filtrage
  • reduce() : Réduction
  • forEach() : Itération
const nombres = [1, 2, 3, 4];
const carres = nombres.map(n => n * n);
const pairs = nombres.filter(n => n % 2 === 0);
const somme = nombres.reduce((a, b) => a + b, 0);

5. Récursivité et Mémoïsation

Récursivité

function factorielle(n) {
    if (n <= 1) return 1;
    return n * factorielle(n - 1);
}
factorielle(5); // 120

Mémoïsation

Cache des résultats pour éviter recalculs.

function memoize(fn) {
    const cache = {};
    return function(arg) {
        if (cache[arg] === undefined) {
            cache[arg] = fn(arg);
        }
        return cache[arg];
    };
}

const fibMemo = memoize(function(n) {
    if (n <= 1) return n;
    return fibMemo(n - 1) + fibMemo(n - 2);
});

Exemple 1 : Calculateur de Panier avec Fonctions d'Ordre Supérieur

Problème : Créer un calculateur de panier avec taxes, réductions et formatage.

Logique : 1. Tableau d'articles avec prix 2. Fonctions de transformation (taxe, réduction) 3. Chaînage avec reduce 4. Formatage monétaire 

<!DOCTYPE html>
<html>
<head>
    <title>Calculateur Panier</title>
    <style>
        .cart { max-width: 500px; margin: 20px auto; font-family: Arial; }
        .item { display: flex; justify-content: space-between; padding: 8px 0; border-bottom: 1px solid #eee; }
        .total { font-weight: bold; margin-top: 15px; text-align: right; }
    </style>
</head>
<body>
    <div class="cart">
        <h2>Panier d'achats</h2>
        <div id="items"></div>
        <div class="total">Total: <span id="total">0.00</span> €</div>
    </div>

    <script>
        const articles = [
            { nom: 'Laptop', prix: 999 },
            { nom: 'Souris', prix: 25 },
            { nom: 'Clavier', prix: 75 },
            { nom: 'Écran', prix: 299 }
        ];

        // Fonctions utilitaires
        const ajouterTaxe = taux => prix => prix * (1 + taux);
        const appliquerReduction = pourcentage => prix => prix * (1 - pourcentage);
        const formaterPrix = prix => prix.toFixed(2);

        // Fonction de calcul
        function calculerTotal(articles, transformations = []) {
            return articles
                .map(a => a.prix)
                .reduce((total, prix) => {
                    let prixModifie = prix;
                    transformations.forEach(fn => {
                        prixModifie = fn(prixModifie);
                    });
                    return total + prixModifie;
                }, 0);
        }

        // Application
        const avecTaxe = ajouterTaxe(0.20); // 20% TVA
        const avecReduction = appliquerReduction(0.10); // 10% promo

        const total = calculerTotal(articles, [avecTaxe, avecReduction]);
        
        // Affichage
        const itemsDiv = document.getElementById('items');
        articles.forEach(a => {
            const div = document.createElement('div');
            div.className = 'item';
            const prixAvecTaxe = formaterPrix(avecTaxe(a.prix));
            const prixFinal = formaterPrix(avecReduction(avecTaxe(a.prix)));
            div.innerHTML = `<span>${a.nom}</span><span>${prixFinal} €</span>`;
            itemsDiv.appendChild(div);
        });

        document.getElementById('total').textContent = formaterPrix(total);
    </script>
</body>
</html>

Résultat attendu : Liste d'articles avec prix après taxe (20%) et réduction (10%). Total calculé automatiquement avec formatage à deux décimales.

Exemple 2 : Générateur de Quiz avec Closures

Problème : Créer un quiz auto-géré avec score, timer et questions aléatoires.

Logique : 1. Banque de questions 2. Closure pour état interne (score, question courante) 3. Méthodes publiques (suivant, vérifier) 4. Timer par question 

<!DOCTYPE html>
<html>
<head>
    <title>Quiz Dynamique</title>
    <style>
        .quiz { max-width: 600px; margin: 30px auto; padding: 20px; font-family: Arial; }
        .question { font-size: 1.2em; margin-bottom: 20px; }
        .options { display: grid; gap: 10px; }
        .option { padding: 12px; border: 1px solid #ddd; border-radius: 4px; cursor: pointer; }
        .option:hover { background: #f0f0f0; }
        .option.correct { background: #d4edda; border-color: #28a745; }
        .option.incorrect { background: #f8d7da; border-color: #dc3545; }
        .controls { margin-top: 20px; text-align: center; }
        #timer { font-weight: bold; color: #e74c3c; }
    </style>
</head>
<body>
    <div class="quiz" id="quiz"></div>

    <script>
        function creerQuiz(questions) {
            let index = 0;
            let score = 0;
            let timer;
            const tempsParQuestion = 15; // secondes

            function melanger(array) {
                return array.sort(() => Math.random() - 0.5);
            }

            function demarrerTimer(callback) {
                let temps = tempsParQuestion;
                document.getElementById('timer').textContent = temps;
                timer = setInterval(() => {
                    temps--;
                    document.getElementById('timer').textContent = temps;
                    if (temps <= 0) {
                        clearInterval(timer);
                        callback(false);
                    }
                }, 1000);
            }

            function afficherQuestion() {
                if (index >= questions.length) {
                    afficherResultat();
                    return;
                }

                const q = questions[index];
                const quizDiv = document.getElementById('quiz');
                
                quizDiv.innerHTML = `
                    <div>
                        <p>Question ${index + 1}/${questions.length}</p>
                        <div class="question">${q.question}</div>
                        <div class="options" id="options"></div>
                        <div class="controls">
                            <span id="timer">${tempsParQuestion}</span>s restantes
                            <button id="next" style="display:none; margin-left:20px;">Suivant</button>
                        </div>
                    </div>
                `;

                const optionsDiv = document.getElementById('options');
                melanger(q.options).forEach((option, i) => {
                    const div = document.createElement('div');
                    div.className = 'option';
                    div.textContent = option;
                    div.onclick = () => verifierReponse(option, div);
                    optionsDiv.appendChild(div);
                });

                demarrerTimer(() => verifierReponse(null, null));
            }

            function verifierReponse(reponse, element) {
                clearInterval(timer);
                const correct = questions[index].reponse;
                
                document.querySelectorAll('.option').forEach(opt => {
                    opt.style.pointerEvents = 'none';
                    if (opt.textContent === correct) {
                        opt.classList.add('correct');
                    } else if (opt === element && reponse !== correct) {
                        opt.classList.add('incorrect');
                    }
                });

                if (reponse === correct) score++;
                document.getElementById('next').style.display = 'inline-block';
                
                document.getElementById('next').onclick = () => {
                    index++;
                    afficherQuestion();
                };
            }

            function afficherResultat() {
                document.getElementById('quiz').innerHTML = `
                    <h2>Quiz terminé !</h2>
                    <p>Score final: ${score}/${questions.length}</p>
                    <p>Pourcentage: ${Math.round((score/questions.length)*100)}%</p>
                    <button onclick="location.reload()">Recommencer</button>
                `;
            }

            return { demarrer: afficherQuestion };
        }

        const questions = [
            {
                question: "Quelle est la capitale de la France ?",
                options: ["Londres", "Berlin", "Paris", "Madrid"],
                reponse: "Paris"
            },
            {
                question: "Combien font 15 * 8 ?",
                options: ["120", "115", "125", "130"],
                reponse: "120"
            },
            {
                question: "Quel est le plus grand océan ?",
                options: ["Atlantique", "Pacifique", "Indien", "Arctique"],
                reponse: "Pacifique"
            }
        ];

        const quiz = creerQuiz(questions);
        quiz.demarrer();
    </script>
</body>
</html>

Résultat attendu : Quiz interactif avec timer 15s par question. Réponses en surbrillance (vert/rouge). Score final avec pourcentage. Bouton rejouer.

Exemple 3 : Système de Cache avec Mémoïsation

Problème : Créer un système de cache pour fonctions coûteuses (API, calculs).

Logique : 1. Fonction de mémoïsation générique 2. Cache par arguments 3. Gestion TTL (temps de vie) 4. Interface de monitoring 

<!DOCTYPE html>
<html>
<head>
    <title>Cache Mémoïsation</title>
    <style>
        .cache-app { max-width: 700px; margin: 30px auto; font-family: Arial; }
        .controls { margin-bottom: 20px; }
        input, button { padding: 8px; margin-right: 10px; }
        .results { margin-top: 20px; }
        .cache-entry { padding: 8px; border-bottom: 1px solid #eee; }
        .hit { color: green; }
        .miss { color: orange; }
        .stats { margin-top: 20px; font-weight: bold; }
    </style>
</head>
<body>
    <div class="cache-app">
        <h2>Système de Cache avec Mémoïsation</h2>
        
        <div class="controls">
            <input type="number" id="input" placeholder="Nombre pour factorielle">
            <button id="calc">Calculer</button>
            <button id="clear">Vider Cache</button>
        </div>
        
        <div class="results">
            <div id="result"></div>
            <div id="cache"></div>
            <div class="stats" id="stats"></div>
        </div>
    </div>

    <script>
        // Fonction de mémoïsation avec TTL
        function memoizeAvecTTL(fn, ttl = 10000) {
            const cache = new Map();
            let hits = 0, misses = 0;

            return {
                executer: function(...args) {
                    const cle = JSON.stringify(args);
                    const maintenant = Date.now();

                    if (cache.has(cle)) {
                        const { valeur, expire } = cache.get(cle);
                        if (maintenant < expire) {
                            hits++;
                            return { valeur, source: 'cache' };
                        } else {
                            cache.delete(cle);
                        }
                    }

                    const valeur = fn(...args);
                    cache.set(cle, { 
                        valeur, 
                        expire: maintenant + ttl 
                    });
                    misses++;
                    return { valeur, source: 'calcul' };
                },
                stats: () => ({ hits, misses, taille: cache.size }),
                vider: () => {
                    cache.clear();
                    hits = 0; misses = 0;
                },
                afficher: () => {
                    const entrees = [];
                    for (const [cle, { valeur, expire }] of cache) {
                        entrees.push({
                            args: JSON.parse(cle),
                            valeur,
                            expire: new Date(expire).toLocaleTimeString()
                        });
                    }
                    return entrees;
                }
            };
        }

        // Fonction coûteuse simulée
        function factorielleLente(n) {
            if (n <= 1) return 1;
            let resultat = 1;
            for (let i = 2; i <= n; i++) {
                resultat *= i;
                // Simulation calcul lourd
                const start = Date.now();
                while (Date.now() - start < 2) {}
            }
            return resultat;
        }

        // Création du cache
        const cacheFactorielle = memoizeAvecTTL(factorielleLente, 30000); // 30s

        // UI
        document.getElementById('calc').addEventListener('click', () => {
            const input = document.getElementById('input');
            const n = parseInt(input.value);
            
            if (isNaN(n) || n < 0) {
                document.getElementById('result').textContent = 'Veuillez entrer un nombre positif';
                return;
            }

            const { valeur, source } = cacheFactorielle.executer(n);
            const classe = source === 'cache' ? 'hit' : 'miss';
            
            document.getElementById('result').innerHTML = 
                `Factorielle(${n}) = <span class="${classe}">${valeur}</span> [${source}]`;
            
            mettreAJourCache();
            mettreAJourStats();
        });

        document.getElementById('clear').addEventListener('click', () => {
            cacheFactorielle.vider();
            mettreAJourCache();
            mettreAJourStats();
        });

        function mettreAJourCache() {
            const cacheDiv = document.getElementById('cache');
            const entrees = cacheFactorielle.afficher();
            
            if (entrees.length === 0) {
                cacheDiv.innerHTML = '<p><em>Cache vide</em></p>';
                return;
            }

            cacheDiv.innerHTML = '<h3>Contenu du cache:</h3>' + 
                entrees.map(e => 
                    `<div class="cache-entry">
                        factorielle(${e.args[0]}) = ${e.valeur} 
                        (expire: ${e.expire})
                    </div>`
                ).join('');
        }

        function mettreAJourStats() {
            const stats = cacheFactorielle.stats();
            document.getElementById('stats').textContent = 
                `Cache: ${stats.taille} entrées | Hits: ${stats.hits} | Misses: ${stats.misses}`;
        }

        // Initialisation
        mettreAJourStats();
    </script>
</body>
</html>

Résultat attendu : Interface calculant factorielle avec cache 30s. Indicateur hit/miss. Affichage contenu cache et statistiques. Bouton vider cache.

Exercice 1 : Générateur de Mot de Passe

Énoncé : Créez une fonction générant un mot de passe aléatoire avec options (longueur, types de caractères).

Démarche : 1. Ensembles de caractères 2. Fonction avec paramètres par défaut 3. Construction chaîne aléatoire 4. Mélange final 

function genererMotDePasse(longueur = 12, 
                           majuscules = true, 
                           chiffres = true, 
                           symboles = true) {
    const minuscules = 'abcdefghijklmnopqrstuvwxyz';
    const maj = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ';
    const nums = '0123456789';
    const sym = '!@#$%^&*()_+-=[]{}|;:,.<>?';
    
    let caracteres = minuscules;
    if (majuscules) caracteres += maj;
    if (chiffres) caracteres += nums;
    if (symboles) caracteres += sym;
    
    let mdp = '';
    for (let i = 0; i < longueur; i++) {
        mdp += caracteres.charAt(Math.floor(Math.random() * caracteres.length));
    }
    
    // Mélange de Fisher-Yates
    return mdp.split('').sort(() => Math.random() - 0.5).join('');
}

// Test
genererMotDePasse(16, true, true, true);

Exercice 2 : Débouncing de Recherche

Énoncé : Créez une fonction de debounce pour limiter les appels API lors de la saisie.

Démarche : 1. Closure avec timer 2. Clear timeout à chaque appel 3. Nouvelle exécution après délai 

function debounce(fn, delai = 300) {
    let timer;
    return function(...args) {
        clearTimeout(timer);
        timer = setTimeout(() => fn.apply(this, args), delai);
    };
}

// Utilisation
const rechercheAPI = debounce(function(terme) {
    console.log('Recherche:', terme);
    // appel API réel ici
}, 500);

document.getElementById('search').addEventListener('input', (e) => {
    rechercheAPI(e.target.value);
});

Exercice 3 : Pipeline de Transformations

Énoncé : Créez une fonction pipe qui enchaîne des transformations sur une valeur.

Démarche : 1. Fonction prenant fonctions en paramètres 2. Reduce pour appliquer séquentiellement 3. Retour fonction finale 

function pipe(...fonctions) {
    return function(valeurInitiale) {
        return fonctions.reduce((valeur, fn) => fn(valeur), valeurInitiale);
    };
}

// Exemple d'utilisation
const traiterTexte = pipe(
    str => str.trim(),
    str => str.toLowerCase(),
    str => str.replace(/\s+/g, '-'),
    str => str + '-traite'
);

traiterTexte('  Bonjour le Monde  '); 
// "bonjour-le-monde-traite"
